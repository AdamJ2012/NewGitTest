Notes:


"-m" in "git commit -m "messagehere"" is short for "message.

"git push" pushes local changes up to GitHub.com.

"git pull" pulls from GitHub.com to local machine.

"git add ." adds everything in your directory.

"git add -A" adds it all

"git" shows the commands and what they do.

"git commit" for a multiline commit message. To get out of it press Escape, then put it :wq, then press enter.

Esc :wq

Putting an empty line inbetween the lines of text here will make it easier to upload these notes to GitHub.com.




e.g.

git add index.html

git commit -m "added index.html"
